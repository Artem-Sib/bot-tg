🤖 Что умеет этот бот:
📜 Факт дня: Каждый день присылает случайный и любопытный факт, чтобы размять мозги.
💡 Умные цитаты: Делится мыслями великих людей для вдохновения и мотивации.
🌤 Погода: Подскажет актуальный прогноз, чтобы ты знал, брать ли зонт.
🎲 Рандомайзер: Помогает принять решение или просто выдает случайное число.
⏰ Напоминания: Не даст забыть о важных делах в течение дня.
🚀 Как запустить:
Клонируй репозиторий.
Установи зависимости: pip install -r requirements.txt.
Добавь свой токен в .env.
Запусти: python main.py
